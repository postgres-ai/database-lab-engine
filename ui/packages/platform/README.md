# Platform Console UI

This is the Platform version of the DBLab Engine UI.

For detailed information about UI development, see the main [CONTRIBUTING.md](../../../CONTRIBUTING.md#ui-development) file. The documentation has been centralized to keep it consistent and up to date.

## Quick reference

### Start the development server
Specify REST API server URL with `REACT_APP_API_SERVER` environment variable.

```bash
npm install
npm run build
npm run start
```

## Troubleshooting

### `meta.json` is missing, what to do?
Run `npm run build`.
